# Agent Skills

Personal agent skills collection.

## Install With Skills CLI

Install `review-plan-implementation`:

```bash
npx skills add https://github.com/supermanyqq/agent-skills --skill review-plan-implementation
```

List available skills:

```bash
npx skills add https://github.com/supermanyqq/agent-skills --list
```

## Included Skills

- `review-plan-implementation`: review whether code implementation strictly matches a plan, spec, acceptance criteria, or task breakdown.
- `ocr`: use macOS Vision framework for local OCR text recognition, scene classification, and object detection. Converts images to structured text for LLMs that don't support multimodal (e.g., DeepSeek).

### Install OCR Skill

```bash
npx skills add https://github.com/supermanyqq/agent-skills --skill ocr
```
